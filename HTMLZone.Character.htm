Les caractères

Description

Cette classe sert à calculer la dimension commune des caractères : on utilise pour cela obligatoirement une police monospace.
Objectif : déterminer la position du curseur dans le texte.

  • largeur caractère = largeur colonne
  • hauteur caractère = hauteur ligne

(La position du curseur dans l'éditeur doit être affiné, la ligne et la colonne devant correspondre à un index dans le texte source)
