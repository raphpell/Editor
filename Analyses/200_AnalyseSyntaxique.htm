Analyse syntaxique

La machine réalisant cette tâche s'appelle le parser.

Elle travail depuis:

  • la liste de léxème obtenue depuis l'analyse lexicale (HTMLElement)
  • un module généré depuis une grammaire.

Le résultat de l'analyse est un arbre syntaxique.

Exemple d'utilisation :

  • Création d'un NFA depuis l'analyse d'une ER.
  • Analyse d'une grammaire
  • Analyse du code générant un module du lexer
  • Analyse des snippets ZenLike
