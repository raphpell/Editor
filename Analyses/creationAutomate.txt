

Un automate possède :
	- I : un état initial
	- F : un ou plusieurs états finaux
	- A : un alphabet
	- S : des états
	- T : des transitions
	- Une liste de nom de token pour les états finaux.
	
I: Etat initial
	Un nombre
	
F: Etats finaux
	Tableau de nombre unique.

A: Alphabet
	Tableau contenant :
		- des caractères
		- des ensembles de caractère négatif ou non
		- le symbole EPSILON
		- le symbole ANY
		
S : Etats ( states en anglais)
	Tableau de nombre unique.

T: Transitions
	Implémentation la plus simple d'une transition : un tableau de 3 à 4 éléments.
		[0] un état initial
		[1] un caractère, ou un ensemble négatif ou non; de l'alphabet
		[2] un état final
		[3] une fonction permettant de transiter (cas ensemble de caractère)
				var f =function( caractere, etatFinal )

Objet Automate

	Fonctions spéciales
		action ( sSymbols, bNegated ) -> Function
			...
		getUniqueID -> Number
			retourne un nombre unique (état)
		setUniqueID -> void
			utilisé dans aggrégation de DFA ?
		wrapper ( s ) -> Automate
			créé un automate depuis une chaine
				Choix possible
					CARACTERES SPECIAUX: '\\n', '\\t', '\\f', '\\r', '\\v'
					ENSEMBLE DE CARACTERES: '\\d', '\\D', '\\s', '\\S', '\\w', '\\W'
				Si chaine.length = 1 : automate à une transition (à un caractère)

	Fonctions retournant une fonction générant un automate unique
		fromChar ( s ) -> Function
			Automate contenant une seule transition (à un caractère).
		fromCharClass ( aSet, bNegated ) -> Function
			Automate contenant une seule transition (à un ensemble de caractère négatif ou non).

	Fonctions générant un automate unique
		Elles ont pour nom :
			CARACTERES SPECIAUX: '\\n', '\\t', '\\f', '\\r', '\\v'
			ENSEMBLE DE CARACTERES: '\\d', '\\D', '\\s', '\\S', '\\w', '\\W'
		Elles n'ont pas d'argument.
