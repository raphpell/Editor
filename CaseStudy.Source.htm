Le texte source

Le texte source est la principale entrée dans l'éditeur : il s'agit d'une simple chaîne de caractère.

Les sauts de ligne

Ils sont définis différemment selon les cas :

  • windows: "\r\n"
  • mac: "\r"
  • unix: "\n"

Pour l'éditeur, il est nécessaire d'utiliser un saut de ligne à un caractère pour avoir la concordance "texte source / sélection native".

  1. Le choix se porte sur les caractères "\n" ou "\r".
  2. Par défaut j'utilise le caractère "\n".

Il est impératif d'identifier le type de sauts de ligne utilisé dans la source auparavant, pour pouvoir le restaurer en cas de sauvegarde par exemple (sauf choix utilisateur).

Les tabulations

Elles sont définies avec le caractère "\t".

  • Leur taille varie selon les navigateurs et les systèmes d'exploitations.
  • Il n'est pas possible de définir la taille des tabulations pour tous les navigateurs.

Avec l'éditeur, on doit pouvoir :

  • définir la taille de tabulations (en nombre de caractère).
  • utiliser des tabulations souples... si on le souhaite.
